# Integrating WakaTime

1. Go to [Projects](https://wakatime.com/projects)
2. Go to repo badge settings

   ![WakaTime Projects](/images/github/wakatime-integration/CleanShot_2022-10-12_at_12.22.562x.png)

3. Turn on the repo badge

   ![Turning on Repo Badge Settings](/images/github/wakatime-integration/CleanShot_2022-10-12_at_12.24.122x.png)

### Missing Repositories

If your GitHub repository is missing, you need to **refresh** them

Go to `https://wakatime.com/projects/<project-name>/edit/github`

![Refresh Repository](/images/github/wakatime-integration/CleanShot_2022-10-12_at_12.30.222x.png)

Then click **refresh repos**
